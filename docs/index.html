Monstar

Demo for a mobile AR game

View on GitHub

This is a demo AR game, which uses the jsfeat CV library for feature tracking.
It was initially meant to be played from a driving car, but the frame rate usually is not high enough :/

Anyway, I think it's a nice demo for using optical flow tracking in implementing AR apps.

Best used on an Android mobile device (iPhones don't support getUserMedia as of this writing).
